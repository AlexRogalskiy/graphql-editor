# Lists and non null

You can use an array type modifier to mark a type as a List, which indicates that this field will return an array of records. In the schema language, this is denoted by wrapping the type in square brackets, [ and ].
You can also set type as non-null by choosing required from menu. In the schema language, this is denoted by exclamation mark “!” after the type name.
