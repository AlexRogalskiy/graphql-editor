# Type Creation

Object type is the most important and most often used element of the whole puzzle, which in the simplest terms is a collection of fields. 
It represents the kind of object you can fetch from the backend. You start with object type and add scalar fields that characterize your object. 
Just right-click mouse to choose type to create a node.
