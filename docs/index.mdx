# Getting started

Getting started with graphql editor is very easy.

## Using GraphQL Editor Component

```sh
$ npm i graphql-editor
```

Navigate to [Editor](./Editor)

## Development

```sh
$ git clone https://github.com/slothking-online/graphql-editor
$ cd graphql-editor
$ npm i
$ npm run start
```

It will run sandboxed environment with basic GraphQL Editor and code.
Feel free to contribute or file an issue.
